## 3. 핵심 기능
- 메인페이지에 대표 상품들을 진열
- 헤더 카테고리에서 전체상품과 각각의 카테고리 이동가
- 리스트에서 퀵카트로 담기 
- 장바구니 페이지에서 담긴 상품 확인
- 주문 페이지에서 구매할 상품 확인 가능
- 이메일 로그인, 가입, 구글 로그인 가능
